# Website Login dan Dashboard Sederhana

Project ini merupakan website sederhana menggunakan HTML, CSS, dan JavaScript yang terdiri dari halaman Login, Register, Lupa Password, dan Dashboard.

## Fitur JavaScript

### 1. Validasi Form Login
- Mengecek input username/email dan password.
- Menampilkan pesan error jika form kosong.
- Menampilkan pesan login berhasil.
- Redirect ke halaman dashboard.

### 2. Validasi Form Register
- Mengecek semua input wajib diisi.
- Mengecek kecocokan password dan konfirmasi password.
- Menampilkan pesan error jika password tidak sama.

### 3. Validasi Lupa Password
- Mengecek input email.
- Menampilkan pesan reset password berhasil dikirim.

### 4. Manipulasi DOM
JavaScript digunakan untuk:
- Mengubah isi teks error/sukses.
- Mengubah warna pesan.
- Menampilkan nama user pada dashboard.

### 5. Local Storage
Data username disimpan agar nama pengguna dapat tampil di halaman dashboard.

### 6. Event JavaScript
Project ini menggunakan beberapa event JavaScript:
- submit
- click
